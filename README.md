# Chess
Chess made using Java AWT and Swing. For AP CS final project

**IMPORTANT: Make sure to put the stockfish executable into the `assets` directory!!**

😎 Features:
- Move highlighting
- Promotion
- Valid move checking
- Basic settings (quit, restart, ai, highlight)
- Really crappy ai 👍
- Check, checkmate
- Stockfish AI
- Auto promotion
